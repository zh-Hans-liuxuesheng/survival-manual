# 简介
在美国很多地方生活必须得开车，这里讨论常见的[买车](buy.md)，[lease车](lease.md)和[卖车](sell.md)的问题。
当你有了一辆车准备开上路前，会有以下几个文件：
- Car Insurance
   - 车险，必须要有车保险才能开车上路
- License Plate
   - 车牌
- Registration
   - 这份文件证明你有到当地交管局注册车辆，需要定期更新
- Title
   - 这份文件直接证明车是你的
- Vehicle Inspection
   - 车检，通常这个是贴在你的车窗上，定期尾气检查时会贴上
